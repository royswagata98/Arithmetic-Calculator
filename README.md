# Arithmetic Calculator

A simple console-based arithmetic calculator built using C++.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Menu-driven interface
- Input validation

## Technologies Used

- C++
- Visual Studio / VS Code

## How to Run

1. Clone the repository.
2. Compile the program.
3. Run the executable.
